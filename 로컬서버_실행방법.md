# 로컬 서버 실행 방법

접속이 안 될 때 아래 방법을 **Cursor 터미널이 아닌, PC에서 직접** 시도하세요.

---

## 방법 1: 배치 파일 (가장 간단)

1. **파일 탐색기**에서 `D:\AI\borahae1\borahae-fan` 폴더로 이동
2. **start-server-and-open.bat** 더블클릭
3. 검은 창이 뜨면 3초 후 브라우저가 **http://localhost:9321/** 로 열림
4. 접속이 안 되면 같은 폴더에서 **PowerShell** 또는 **명령 프롬프트**를 열고 아래 방법 2 실행

---

## 방법 2: Cursor 터미널에서 서버 실행 (권장)

**Cursor 하단 터미널**에서 아래 중 하나만 실행하세요.

**A) Node (Python 없이):**
```bash
npm run serve
```
→ **http://localhost:9321/** 로 접속

**B) Python:**
```powershell
cd "D:\AI\borahae1\borahae-fan"
python -m http.server 9321
```
→ **http://localhost:9321/** 로 접속

터미널에 주소가 뜨면 **Chrome/Edge**에서 그 주소를 **직접 입력**하세요.

---

## 방법 3: Node 서버 (API 사용 시)

프로젝트에 Node 서버가 있습니다. 포트 **8000** 사용:

```powershell
cd "D:\AI\borahae1\borahae-fan"
npm run dev
```

브라우저에서 **http://localhost:8000/** 입력.

---

## 방법 4: Live Server (Cursor/VS Code)

1. Cursor에서 **확장(Extensions)** 설치: **Live Server**
2. 왼쪽 파일 탐색기에서 **index.html** 우클릭
3. **Open with Live Server** 선택
4. 브라우저가 자동으로 열리며 보통 **http://127.0.0.1:5500/** 사용

---

## 접속이 안 될 때 확인

- **방화벽**: Windows 방화벽에서 Python 또는 Node 허용 여부 확인
- **포트 충돌**: 9321 대신 `python -m http.server 3000` 처럼 다른 포트 시도
- **브라우저**: **Chrome, Edge** 등 Cursor 밖의 브라우저에서 **http://localhost:9321/** 직접 입력
