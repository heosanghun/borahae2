# 보라해 홈페이지 배경음(BGM) 적용 제안

## 1. 왜 “선택형” 배경음인가

- **자동 재생 소리**는 많은 사용자가 불쾌해하고, 접근성 가이드라인(자동 재생 제한)에도 맞춰야 합니다.
- **기본은 음소거**, 사용자가 **“BGM 켜기”**를 눌렀을 때만 재생하는 방식을 추천합니다.
- 이미 배경 **영상(muted)**이 있으므로, BGM은 “선택적 즐거움”으로 두는 것이 안전합니다.

## 2. 추천 구현 방식

| 항목 | 제안 |
|------|------|
| **재생** | 전역 `<audio>` 1개, **루프 재생** |
| **시작** | **정지 상태** → 사용자가 버튼으로 재생 시작 |
| **위치** | 우측 하단 고정 또는 네비게이션 옆 **BGM 토글 버튼** (🔇 / 🔊) |
| **볼륨** | 기본 20~30% 수준으로 설정, 필요 시 작은 슬라이더 추가 |
| **로딩** | `preload="none"` 또는 `metadata` → 버튼 클릭 시 로드해 첫 진입 부담 감소 |

## 3. 음원 파일

- `music` 폴더의 MP3를 그대로 사용할 수 있습니다.
- 구현에서는 **`music/bgm.mp3`** 경로를 사용합니다.  
  - **방법 A**: MP3 파일 이름을 **`bgm.mp3`**로 바꿔 주세요.  
  - **방법 B**: `index.html` 상단의 `window.BGM_URL = 'music/bgm.mp3';` 를 실제 파일명으로 수정하세요 (예: `'music/보라해 시그널.mp3'`).
- 적용된 동작: 우측 하단 **🔇/🔊 버튼** 클릭 시 재생·일시정지, 볼륨 25%, 루프 재생, 선택 시 localStorage에 저장해 다음 방문 시 유지됩니다.

## 4. 참고 자료 활용

- **가사.tex**  
  - “내 탄생뮤직 가사” 팝업이나 별도 “이 곡의 가사” 영역에 넣어, BGM과 함께 감상할 수 있게 할 수 있습니다.
- **background style.jpg**  
  - BGM 섹션/모달의 분위기 참고용 이미지로 사용하거나, 테마 설명용으로 활용할 수 있습니다.

## 5. 추가 아이디어 (선택)

- **특정 섹션만 BGM**  
  - 예: 메인 히어로 구간에서만 재생, 스크롤로 다른 섹션 진입 시 일시정지/볼륨 다운.
- **“이 곡 가사 보기”**  
  - BGM 토글 옆에 작은 링크를 두고, 클릭 시 가사.tex 내용을 팝업/패널로 표시.
- **저사양/모바일**  
  - `preload="none"`으로 두고, 재생 요청 시에만 로드해 트래픽·배터리 부담을 줄일 수 있습니다.

---

**요약**: BGM은 **사용자 선택으로만 재생**하고, **`music/bgm.mp3`**(또는 설정한 경로)를 루프 재생하며, 우측 하단 등에 **토글 버튼**을 두는 구성을 추천합니다.
