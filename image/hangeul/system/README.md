# 한글 그리드 시스템 참고 (image/hangeul/system)

## 사랑의 인사 1. 한글 그리드 이미지

- **참고 스타일**: 「사랑의 인사 (Salut d'Amour), Op.12」 1~16마디를 한글 자모 그리드로 표현한 형식입니다.
- **구성**: 마디 번호(열) × 4역할(행) — **구조(베이스)**, **멜로디(장식)**, **반주 오른손**, **반주 왼손**. 각 칸은 음계→한글 자모(도레미파솔라시↔ㄱㄴㄷㄹㅁㅂㅅ 등) 규칙으로 채워집니다.
- **매직샵 동작**: 사용자가 악보(MIDI·PDF·이미지)를 업로드하면, 이 **그리드 스타일**을 참고하여 **악보의 마디 순서에 맞춰** 같은 형식의 한글 그리드 이미지를 생성합니다.  
  - 악보 이미지 → Gemini로 조(key) + 마디별 음(B, M, RAcc, LAcc) 추출 → `notesToJamoBars()`로 한글 자모 변환 → `generateArchitectureImage()`로 그리드 PNG 생성.

## 폴더 구조

- **information/** — 매직샵 우측 캐러셀용 시스템 설명 이미지 (0.svg~6.svg 등).
- (선택) **사랑의 인사 1. 한글 그리드** 참고용 PNG를 이 폴더에 두면, 동일 스타일로 악보에 맞춘 그리드 생성에 활용할 수 있습니다.

## 악보에 맞추어 생성

- 그리드의 **열(마디)** 은 악보에서 보이는 **마디 순서**와 1:1 대응합니다. (bar 1 = 첫 마디, bar 2 = 두 번째 마디, …)
- 업로드한 악보가 16마디 미만이면 분석된 만큼만 채우고, 나머지는 비우거나 샘플로 보완합니다.
