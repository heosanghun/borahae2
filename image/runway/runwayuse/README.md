# 갤러리 배경 이미지

갤러리에서 사용할 이미지를 이 폴더에 넣어 주세요.

**광화문 공연장소 (a, b, c)**  
- `a.jpg` 또는 `a.png` → 광화문 공연장소 1  
- `b.jpg` 또는 `b.png` → 광화문 공연장소 2  
- `c.jpg` 또는 `c.png` → 광화문 공연장소 3  

**뮤직비디오 장소 (1, 2, 3)**  
- `1.jpg` 또는 `1.png` → 1 뮤직비디오 장소  
- `2.jpg` 또는 `2.png` → 2 뮤직비디오 장소  
- `3.jpg` 또는 `3.png` → 3 뮤직비디오 장소  

**뮤직비디오 장소 (4, 5, 6)**  
- `11.jpg` 또는 `11.png` → 4 뮤직비디오 장소  
- `22.jpg` 또는 `22.png` → 5 뮤직비디오 장소  
- `33.jpg` 또는 `33.png` → 6 뮤직비디오 장소  

파일이 없어도 기본 썸네일(a.svg ~ c.svg, 1.svg ~ 3.svg, 11.svg ~ 33.svg)이 표시됩니다.  
실제 사진을 넣으면 자동으로 교체됩니다.

**권장 사진 크기**  
- 한 변 1200px 이하, 파일 크기 1MB 이하를 권장합니다.  
- 너무 큰 사진(수 MB)은 로딩이 느리거나 일부 환경에서 타임아웃이 날 수 있습니다.
