# 뮤직비디오 대표 갤러리 배경 (인물 제거본)

- 이 폴더의 이미지는 **인물·텍스트가 제거된 배경만** 사용합니다.
- 갤러리: **광화문 공연 장소**, **다이너마이트배경**, **버터배경** 세 가지를 선택할 수 있습니다.
- PNG가 없으면 같은 이름의 `.svg` 썸네일이 표시됩니다. 실제 배경용 PNG를 넣으면 자동으로 사용됩니다.

## 배경 이미지 생성 방법

1. `image/runway/` 폴더에 원본 이미지(.jpg 또는 .png)를 넣습니다.
2. 터미널에서 실행:
   ```bash
   set GEMINI_API_KEY=your_key
   node scripts/generate-runway-backgrounds.js
   ```
3. 생성된 이미지가 이 폴더에 저장되고, `list.json`이 갱신됩니다.
4. **다이너마이트**, **버터** 스타일 배경은 `dynamite.png`, `butter.png`로 추가하면 갤러리에 적용됩니다.
