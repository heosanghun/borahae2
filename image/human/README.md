# image/human – 샘플 테스트용

## 샘플 얼굴 (런웨이)

- **여자**: 이 폴더에 `soave.jpg` 또는 `soave.png` 를 넣으면 샘플 얼굴 선택에 사용됩니다.
- **남자**: 이 폴더에 `ian.jpg` 또는 `ian.png` 를 넣으면 샘플 얼굴 선택에 사용됩니다.

파일이 없으면 기본 이미지(Unsplash)로 대체됩니다.

---

이 폴더의 **sample.txt** 에 적힌 이름·생년월일이 페이지 로드 시 자동으로 아래 입력란에 적용됩니다.

- 원클릭 런웨이: 이름, 생년월일
- 이름 속에 숨겨진 영혼의 자취: 이름
- 소울 컬러: 생년월일

## sample.txt 형식

**한글 형식 (권장)**

```
이름 : 보라해
생년월일: 2000년 3월 21일
```

**키=값 형식**

```
name=테스트
birth=1995-06-15
```

생년월일은 `yyyy-mm-dd`, `mm/dd/yyyy`, `yyyy년 n월 n일` 모두 가능합니다.

앞으로 샘플 테스트할 때는 이 파일만 수정하면 됩니다.
