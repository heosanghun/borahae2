# 얼굴 샘플 (Face samples)

스타일링 Step 2에서 "얼굴 샘플"로 선택할 수 있는 이미지를 이 폴더에 넣으세요.

- **list.json**: 사용할 이미지 파일 이름 목록 (JSON 배열)
  - 예: `["face1.jpg", "face2.jpg", "sample.png"]`
- 이 폴더에 해당 파일을 두면, 앱에서 썸네일로 보이고 클릭 시 얼굴 사진으로 적용됩니다.

**원클릭 런웨이** (당신만의 런웨이 한 편):
- **face1.jpg**: 여자 샘플
- **face2.jpg**: 남자 샘플
두 파일을 넣으면 갤러리에서 여자/남자 선택 후 배경만 고르고 런웨이를 만들 수 있습니다.
- 샘플 사진 자동 저장: 프로젝트 루트에서 `node scripts/download-face-samples.js` 실행 시 위 두 파일이 Unsplash 샘플로 저장됩니다.
