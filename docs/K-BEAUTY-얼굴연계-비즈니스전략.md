# 얼굴 업로드 기반 K-뷰티 상품 연계 비즈니스 전략

> **타깃**: 외국인, K-뷰티 화장품에 관심 있는 고객  
> **목표**: 얼굴 입력 → 분석 → 얼굴에 맞는 K-뷰티 상품·광고 연계

---

## 1. 가치 제안 (Value Proposition)

| 대상 | 니즈 | 우리 솔루션 |
|------|------|-------------|
| **해외 사용자** | 한국 화장품을 사고 싶은데 톤·제품을 모름 | 얼굴 업로드 → 퍼스널 컬러·피부톤 분석 → **맞는 K-뷰티 제품 추천** |
| **K-뷰티 관심층** | 나에게 맞는 제품·색상을 쉽게 찾고 싶음 | AI 분석 + Virtual Try-On으로 **나만의 컬러·스타일** → 제품 매칭 |
| **화장품 브랜드** | 타깃 맞는 광고·프로모션 노출 | **얼굴/컬러 기반 세그먼트**에 맞춤 배너·딥링크 |

**한 문장**: “얼굴 한 장으로 퍼스널 컬러와 스타일을 보고, 그에 맞는 K-뷰티 상품과 연결한다.”

---

## 2. 사용자 플로우 (기존 BORAHAE와 연계)

```
[1. 기본 정보] 이름·생년월일·선호
       ↓
[2. 신체 측정] 얼굴 사진 업로드 + 키·몸무게  ← ★ 입력 포인트
       ↓
[3. 스타일 취향] 로맨틱/엘레강트/캐주얼 등
       ↓
[4. 컬러 분석] 피부톤·언더톤(웜/쿨/뉴트럴) → 퍼스널 컬러(봄웜·여름쿨 등)
       ↓
[5. AI 분석] 퍼스널 컬러 + 스타일 → "나만의 컬러 한글 친구" + 추천 스타일
       ↓
[6. Virtual Try-On] 전신 코디·메이크업 시뮬레이션
       ↓
[연계] "이 톤에 맞는 K-뷰티 추천" → 제품 카드·광고·구매 링크
```

- **얼굴 업로드**는 이미 Step 2에 있음 → 여기서 나온 **피부톤·퍼스널 컬러·스타일**을 K-뷰티 연계의 **입력 데이터**로 사용.

---

## 3. 화장품·광고 연계 비즈니스 전략

### 3-1. 수익화 구조

| 방식 | 설명 | 적용 예 |
|------|------|----------|
| **제휴/어필리에이트** | 추천 제품 클릭 시 구매 시 수수료 | 올리브영, 쿠팡, 아마존, 글로우픽 등 K-뷰티 채널 |
| **브랜드 광고** | 톤·컬러·스타일 세그먼트에 맞춤 배너/동영상 | “여름쿨 톤 사용자에게 라벤더 립 추천” |
| **스폰서드 추천** | “이 분석 결과에 어울리는 OOO 제품” 노출 | 브랜드가 특정 퍼스널 컬러 구간 스폰서 |
| **데이터 인사이트** | 익명 집계 데이터로 브랜드에 리포트 (선택) | “해외 사용자 중 쿨톤 비율 40%” 등 |

### 3-2. 연계 시점 (UX)

1. **컬러 분석 직후**  
   - “여름쿨 톤이에요. 이런 립·블러셔가 잘 어울려요.” → 2~3개 제품 카드 + 링크.
2. **Virtual Try-On 직후**  
   - “지금 코디에 이 메이크업/립 색상은 어때요?” → 시뮬레이션과 동일 톤 제품 추천.
3. **결과 공유 시**  
   - “내 퍼스널 컬러 결과를 공유하고 맞춤 K-뷰티 보기” → 랜딩/이벤트 페이지로 유도.

### 3-3. 외국인 타깃 전략

- **다국어**: 분석 결과·추천 문구를 영어(필요 시 일본어·중국어) 지원 → “Your tone: Summer Cool. These K-beauty products match you.”
- **글로벌 구매 경로**: 해외 배송 가능 채널(아마존, YesStyle, 스타일코리아 등) 링크 연동.
- **콘텐츠**: “K-beauty for your skin tone” 짧은 가이드 + 추천 제품 리스트로 검색·SNS 유입 활용.

### 3-4. 광고 연계 시 주의사항

- **동의**: “맞춤 제품·광고를 위해 분석 결과를 활용합니다” 등 명시적 동의.
- **개인정보**: 얼굴·분석 결과는 서버 저장 최소화 또는 익명화 후 집계만 사용.
- **노출 구분**: “추천” vs “광고/스폰서” 라벨을 구분해 신뢰 유지.

### 3-5. 정책·운영 보완 (상세는 전문가 분석 문서 참고)

- **동의 시점**: Step 2 완료 시 또는 Step 4(컬러 분석) 진입 시, "맞춤 K-뷰티 추천 및 제휴 링크 안내에 톤·스타일 결과를 활용합니다" 문구 + 동의 체크 권장.
- **광고·제휴 라벨**: 한국은 "광고"·"제휴 링크 포함", 해외는 "Sponsored"·"Affiliate link" 등 노출. (관련 법령·플랫폼 정책 확인 필요.)
- **현재 한계**: 톤 분석은 사용자가 Step 4에서 직접 선택하는 방식이며, 얼굴 사진 기반 자동 톤 추정은 추후 단계에서 검토.

## 4. 단계별 실행 로드맵

| 단계 | 내용 | 비고 |
|------|------|------|
| **Phase 1** | 컬러 분석 결과 화면에 “이 톤에 어울리는 K-뷰티” 1~2개 슬롯 추가 (하드코딩/수동 추천) | 빠른 검증 |
| **Phase 2** | 퍼스널 컬러·피부톤별 제품 DB(이미지, 링크, 톤 태그) 구축 + 매칭 로직 | 자동 추천 |
| **Phase 3** | 어필리에이트/광고 파트너 계약 후 링크·배너 연동 | 수익화 |
| **Phase 4** | Virtual Try-On 결과와 동일 톤 메이크업 제품 연동 + 다국어 | 외국인 확대 |
| **Phase 1.x** | 얼굴 사진 기반 톤 자동 추정 검토 (현재는 수동 선택) | 분석 정확도·외국인 UX 개선 |

※ **문제점·보완점·제안점** 전체 분석은 `K-BEAUTY-전문가-분석-및-개선안.md` 참고.

## 5. 요약: “최고의 전략” 한 줄

**“얼굴 업로드 → 퍼스널 컬러·스타일 분석이라는 재미·가치를 먼저 주고, 그 결과에 맞는 K-뷰티 상품과 광고만 골라서 보여준다.”**

- **고객**: 나에게 맞는 제품을 쉽게 발견.
- **브랜드**: 타깃 맞는 광고·프로모션.
- **플랫폼**: 제휴 수수료·광고료로 수익화.

이 문서는 기획·파트너 제안·내부 정렬용으로 활용하시면 됩니다.  
구체적인 UI 위치나 API 설계가 필요하면 별도 문서로 이어서 정리할 수 있습니다.
