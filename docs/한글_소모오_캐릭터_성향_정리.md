## 성향을 4군·7개씩으로 나눈 참고 (순서 강제 아님)

※ **한글 캐릭터를 ㄱ→ㄴ→ㄷ… 순서로 꼭 묶을 필요는 없습니다.**  
  빨주노초파남보·도레미파솔라시 연관은 유지하되, 7색·7음과 한글 매핑은 **성향·무드에 맞게** 자유롭게 정하면 됩니다.

- **군1 (자음 1~7)**: ㄱ 고롱 → … → ㅅ 소롱 → **발명·표현·돌봄·요리·공예·탐구·감성(시)**
- **군2 (자음 8~14)**: ㅇ 오롱 → … → ㅎ 호롱 → **유쾌·상상·춤·경계·논리·탐정·보호**
- **군3 (모음 1~7)**: ㅏ 아롱 → … → ㅜ 우롱 → **그림·전달·정원·치유·마술·동물·기록**
- **군4 (모음 8~14)**: ㅠ 유롱 → … → ㅖ 예롱 → **탐험·명상·재단·교육·사진·안내·연주**

- **자음 14**: 기능·구조·재능 유형 (발명, 가수, 도우미, 요리, 공예, 천문, 시, 개그, 상상, 댄서, 파수, 논리, 탐정, 힘).
- **모음 14**: 감성·숨결·역할 (화가, 전령, 정원, 치유, 마술, 동물, 기록, 탐험, 명상, 재단, 교육, 사진, 길잡이, 연주).
- **7색(빨주노초파남보) ↔ 7음(도레미파솔라시)** 연관은 유지해도 되고, **한글 28개는 순서가 아니라 성향**으로 7색·7음에 매핑하면 됨. (예: 빨강=열정 → 초롱·호롱 등, 보라=감성 → 소롱·예롱 등)
