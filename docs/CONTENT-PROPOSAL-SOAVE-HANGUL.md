# 아미팬의 공간에 소아베·한글 캐릭터 녹이기 — 콘텐츠 제안서

**전제**:  
- 보라해는 **아미팬의 공간**. 과한 캐릭터 노출·스토리 과잉은 부담을 줄 수 있음.  
- **조심스럽게** = “있으면 좋은 맛”으로 넣고, 팬이 몰입할 수 있는 선택권을 유지하는 쪽을 권함.

**목표**:  
- **소아베**(ARMY 주인공)와 **한글 소모오 캐릭터 28종**을 홈페이지 콘텐츠에 자연스럽게 녹여, “우리 공간의 얼굴”로 자리 잡게 하기.

---

## 1. 원칙 (조심스럽게 녹일 때 기준)

| 원칙 | 설명 |
|------|------|
| **선택 가능** | 스토리/캐릭터를 싫어하는 팬은 “그냥 서비스만” 쓸 수 있게, 캐릭터는 **강제가 아닌 강화**로만 사용. |
| **아미 시선 유지** | 소아베는 “팬 대표 캐릭터”이지, 아티스트를 대변하지 않음. 문구는 항상 **팬(우리)** 시점. |
| **점진 도입** | 한 번에 전면 개편보다 **문구 → 일부 시각 → 섹션 추가** 순으로 단계 적용. |
| **기능과 연결** | 캐릭터가 “그냥 예쁜 그림”이 아니라 **스타일링·컬러·굿즈·채팅** 등 기능과 연결되면 납득도 상승. |

---

## 2. 구역별 제안 (어디에 어떻게 넣을지)

### 2.1 히어로(상단) — “소아베가 우리를 맞이한다”

**현재**: “보라해 · 팬이 만드는 보라빛 세상” + K-pop 팬 라이프스타일 플랫폼 소개.

**제안**  
- 문구만 살짝 **소아베 시점**으로 바꿔서 “팬이 말하는 느낌”을 줌.  
  - 예 (한 줄 추가):  
    - `보라해, 팬이 만드는 보라빛 세상`  
    - → 그 아래 또는 태그라인: **“소아베와 함께하는, 나만의 보라빛”** (선택 문구)  
  - 또는 **히어로 설명문** 한 문장:  
    - “K-pop을 사랑하는 당신을 위한 공간. 소아베가 안내하는 AI 스타일링, 굿즈, 커뮤니티까지.”  
- **이미지**:  
  - 1단계: 기존 폰 목업·도형 유지.  
  - 2단계: 폰 화면 옆이나 하단에 **소아베 일러스트 1장**(웃는 정면)을 작게 배치해 “이 공간의 주인공”을 시각적으로만 암시.  
- **주의**: 히어로는 짧게. “소아베” 이름만 노출해도 충분하고, 긴 스토리는 아래 섹션으로.

---

### 2.2 서비스(Features) 카드 — “한글 캐릭터가 섹션 가이드”

**현재**: AI 스타일링, 굿즈샵, 커뮤니티, 이벤트, 팬 콘텐츠, 멤버십 6개 카드.

**제안**  
- 각 카드에 **한글 캐릭터 1명**을 “이 서비스를 대표하는 친구”로 붙임.  
  - 성향이 맞는 캐릭터만 골라서 **순서 무관** (문서 “한글은 순서로 묶을 필요 없음” 반영).  

| 서비스 | 추천 캐릭터 | 연결 이유 (성향) |
|--------|-------------|------------------|
| AI K-pop 스타일링 | **이롱** (재단사) | 옷·맞춤·스타일 → “딱 맞는 코디” |
| 굿즈샵 | **아롱** (화가) | 색·디자인·팬메이드 감성 |
| 팬 커뮤니티 | **도롱** (도우미) | 친절·배려·함께하는 공간 |
| 이벤트 기획 | **야롱** (전령사) | 소식 전달·빠른 실행 |
| 팬 콘텐츠 | **소롱** (시인) / **얍롱** (사진가) | 감성·표현·기록 |
| 멤버십 | **에이롱** (길잡이) | 안내·플랜 선택 도움 |

**구현**  
- 카드 상단 또는 아이콘 자리에 **캐릭터 작은 이미지**(자음14·모음14 에셋) + 이름(예: “이롱이 알려주는 스타일링”).  
- 카드 설명문은 **기존 문장 유지**하고, 부제만 “○○이가 안내하는 [서비스명]” 정도로 추가해도 됨.  
- 1단계는 **텍스트만** (“이롱이 알려주는 AI 스타일링”) 적용하고, 2단계에서 이미지 추가.

---

### 2.3 AI 스타일링 섹션 — “소아베가 격려하는 코디 여정”

**현재**: 3단계(프로필 분석 → K-pop 스타일 매칭 → 보라해 코디 완성) + CTA.

**제안**  
- **헤딩/설명**에 소아베를 한 번만 자연스럽게 넣기.  
  - 예:  
    - “3단계로 완성되는 K-pop 감성 코디”  
    - → “소아베와 함께, 3단계로 완성하는 나만의 K-pop 코디” (또는 부제로 “소아베가 응원하는 코디 여정”)  
- **스텝 설명**은 기능 중심 유지.  
  - 선택: 각 스텝 옆에 **말풍선 1줄** 형태로 소아베 한 마디만.  
    - Step 1: “사진 한 장이면 돼, 함께 볼게!”  
    - Step 2: “이 무드에 딱 맞는 스타일 찾아볼게.”  
    - Step 3: “보라빛으로 물든 너만의 코디 완성!”  
- **퍼스널 컬러 결과 화면**(스타일링 모달 내)  
  - 결과 카드 하단에 **“나만의 컬러에 어울리는 한글 친구”** 1명 노출.  
  - 7색 매핑된 한글 캐릭터 1명(이미지+이름+한 줄 성향)으로 “이 컬러의 무드와 잘 맞는 재능·성향”을 안내 → **나만의 재능**으로 자연스럽게 이어지게.

---

### 2.4 보라빛 응원봉 — “한글 친구들이 단계별로 등장”

**현재**: STEP 1~4 (이름 새기기 → 컬러 선택 → AI 디자인 → 공유).

**제안**  
- 각 STEP에 **한글 캐릭터 1명**을 “이 단계를 대표하는 친구”로만 표시.  
  - STEP 1 (마음 새기기): **소롱** (시인) — 말과 마음을 담는 이미지.  
  - STEP 2 (7컬러 선택): **아롱** (화가) — 색과 감성.  
  - STEP 3 (AI 디자인): **고롱** (발명가) — 만들기·창조.  
  - STEP 4 (공유): **야롱** (전령사) — 전달·나누기.  
- 모달 상단 스텝 인디케이터 옆이나 제목 아래에 **캐릭터 작은 아이콘 + 이름**만 넣어도 “우리 공간의 친구들이 도와준다”는 인상만 줄 수 있음.

---

### 2.5 굿즈샵 — “한글 캐릭터가 카테고리 안내”

**현재**: 의류, 에코백, 폰케이스, 키링, 문구·다이어리, 스티커, 시계 등 카테고리.

**제안**  
- 카테고리별로 **한글 캐릭터 1명** 연결 (선택).  
  - 예: 문구·다이어리 → **우롱** (기록가), 스티커·데코 → **아롱** (화가), 의류 → **이롱** (재단사).  
- “○○이가 추천하는 [카테고리]” 같은 짧은 라벨만 붙여도 톤이 통일됨.  
- 1단계는 **가장 인기 카테고리 2~3개**에만 적용해 테스트.

---

### 2.6 채팅 위젯 — “소아베가 말하는 AI”

**현재**: “보라해 AI 스타일리스트” + 아바타 “AI” 텍스트.

**제안** (스토리텔링 계획서 Phase 1과 동일)  
- **아바타**: “AI” → **소아베 얼굴 이미지**(소모오 소아베 에셋).  
- **헤더**: “보라해 AI 스타일리스트” → **“소아베”** 또는 “소아베와 대화”.  
- **웰컴 메시지**:  
  - “저는 보라해 AI 스타일리스트입니다…”  
  - → “나는 보라해를 사랑하는 아미, 소아베야. 코디나 스타일 고민 같이 나눠 보자.”  
- **말풍선**: AI 응답 시 **말풍선 + 소아베 얼굴** 레이아웃으로 “소아베가 말하는 것처럼” 연출.  
- 이렇게 하면 **아미팬의 공간**에서 “우리 중 한 명(소아베)이 말해 주는 AI”라는 인식이 생김.

---

### 2.7 퍼스널 컬러 결과 + “나만의 재능” (한글 캐릭터 성향)

**현재**: 퍼스널 컬러 시즌, 설명, 팔레트, 7컬러 음악 추천.

**제안**  
- 결과 카드에 **“나만의 컬러에 어울리는 한글 친구”** 블록 1개 추가.  
  - 7색(빨주노초파남보) 중 대표 1색에 **성향이 맞는 한글 캐릭터 1명** 매핑.  
  - 노출 요소: 캐릭터 이미지(작게) + 이름(예: “초롱”) + 역할(예: “댄서”) + 한 줄 문구(예: “열정과 리듬이 있는 너에게 어울리는 친구예요. 춤처럼 에너지를 발산해 보세요.”).  
- “나만의 재능” 문구는 **캐릭터 성향표**(`한글_소모오_캐릭터_성향_정리.md`) 기반으로 7색별 1~2문장만 정의하면 됨.  
- **순서 강제 없음**: 색–캐릭터 매핑은 성향으로만 정하고, ㄱㄴㄷ 순서는 사용하지 않음.

---

### 2.8 푸터·법적 문구

**유지**:  
- “팬 문화·팬 활동 지원”, “아티스트 권리는 권리자에게”, “수익은 플랫폼 운영용” 등 기존 문구 유지.  
- 소아베·한글 캐릭터는 **플랫폼·스토리 콘텐츠**로만 표기하고, “공식 캐릭터”처럼 과도하게 주장하지 않음.

---

## 3. 도입 단계 제안 (우선순위)

| 단계 | 내용 | 리스크 | 비고 |
|------|------|--------|------|
| **1단계** | 채팅 위젯만 소아베화 (아바타·헤더·웰컴 문구·말풍선) | 낮음 | 한 곳만 바꿔도 “주인공” 체감 가능 |
| **2단계** | 히어로·AI 스타일링 섹션 문구에 소아베 1~2문장, 퍼스널 컬러 결과에 “한글 친구 1명” 블록 | 낮음 | 문구·블록만 추가 |
| **3단계** | 서비스(Features) 카드에 “○○이가 알려주는” + 캐릭터 이름(이미지 선택) | 중간 | 일관된 톤, 이미지는 에셋 준비 후 |
| **4단계** | 응원봉 STEP별·굿즈 카테고리별 한글 캐릭터 라벨, 히어로에 소아베 비주얼 | 중간 | 전반적 “한글 친구들” 느낌 |
| **5단계** | 별도 “스토리” 또는 “소아베와 한글 친구들” 소개 페이지 (선택) | 검토 | 스토리 좋아하는 팬용 심화 콘텐츠 |

---

## 4. 문구 톤 가이드 (소아베·한글 캐릭터)

- **소아베**:  
  - 1인칭 “나”, “우리”(아미)와 한 팀.  
  - 예: “나도 콘서트 코디 고민 많아. 같이 찾아보자.” / “이 컬러, 너한테 잘 어울릴 것 같아.”  
  - 격려·동행 느낌, 과한 캐릭이나 스토리 스포는 자제.  

- **한글 캐릭터**:  
  - “○○이가 알려주는 / 추천하는 / 안내하는” 정도로 **기능·섹션과만** 연결.  
  - 캐릭터 이름 + 역할(재능) 한 단어만 노출해도 성향이 전달됨.  
  - 예: “이롱(재단사)이 알려주는 스타일링”, “초롱(댄서)과 맞는 컬러예요.”

---

## 5. 정리

- **소아베**: 히어로·스타일링·**채팅**에서 “아미 대표 주인공”으로, 말과 얼굴(아바타·말풍선)로만 조심스럽게 노출.  
- **한글 캐릭터 28종**: **순서로 묶지 않고**, 서비스·기능·7컬러 결과와 **성향으로만** 연결해 “이 공간을 도와주는 친구들”로 위치시키기.  
- **아미팬의 공간**을 지키려면: 캐릭터를 “필수”가 아니라 “있으면 더 좋은 맛”으로 두고, 1단계(채팅 소아베화)부터 적용한 뒤 반응을 보며 2~4단계를 확대하는 것을 권합니다.

이 제안서를 기준으로, 먼저 적용할 구역(예: 채팅만 / 채팅 + 퍼스널 컬러 한글 친구)을 정하면 단계별로 문구·구조 초안을 구체화할 수 있습니다.
