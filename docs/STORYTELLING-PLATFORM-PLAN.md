## 2. 방향 설정 (토의 포인트)

- **스토리와 홈의 관계**
  - A) 메인 화면부터 스토리(장면·내레이션)가 흐르고, 서비스(스타일링·굿즈·멤버십)는 “스토리 속 소아베가 경험하는 세계”로 묶기  
  - B) 기존 홈 구조는 유지하고, **채팅/말풍선·특정 섹션만** “소아베가 말하는 톤”으로 바꾸기  
  - C) 별도 **「스토리」 메뉴**를 두고, 그 안에서만 「AI, 마음을 스케치하다」 연재·읽기 형태로 제공  

- **소아베 정체성**
  - BTS를 좋아하는 **ARMY**이자 스토리의 **주인공**.
  - 채팅/말풍선에서는 “소아베가 사용자에게 말하는 것처럼” 보이게 할지, “AI가 소아베 톤으로 답하는 것처럼” 할지 결정 필요.

- **소모오 캐릭터(자음14·모음14 = 28한글) 역할**
  - 스토리 내 등장인물, 섹션별 가이드, 이모티콘·스티커 등으로만 쓸지,  
  - 또는 “소아베와 친구들”처럼 플랫폼 전반의 얼굴로 쓸지 토의.
  - **나만의 컬러 → 28 한글 캐릭터 성향 → 나만의 재능 발견** 연계는 `docs/IDEA-EVALUATION-28HANGUL-COLOR-TALENT.md` 참고.

## 3. 단계별 개발 계획 (안)

### Phase 0: 준비 (토의와 자료 정리)
- [ ] PDF **「AI, 마음을 스케치하다」** 1권 내용 요약·장면 구분·주요 대사 정리 (수동 또는 도구 활용).
- [ ] **주인공 소아베** 설정 확정: 이름, 성격 한 줄, “말하는 톤” 가이드 문구.
- [ ] 소모오 캐릭터 중 **우선 사용할 이미지/영상 목록** 선정 (소아베 얼굴 1~3종, 자음 캐릭터는 필요 시만).
- [ ] **스토리와 홈 관계** 방향 결정: A/B/C 중 선택 또는 조합.

### Phase 1: 말풍선·채팅을 “소아베가 말하는” 형태로 변경 (우선 적용)
- [ ] **채팅 위젯**
  - 플로팅 버튼·채팅 열었을 때 **아바타**를 “AI” 텍스트 → **소아베 캐릭터 얼굴** 이미지로 교체.
  - 채팅 헤더 타이틀: “보라해 AI 스타일리스트” → **“소아베”** 또는 “소아베와 대화” 등으로 변경 (i18n 반영).
- [ ] **웰컴 메시지**
  - “저는 보라해 AI 스타일리스트입니다…” → **소아베 1인칭 톤**으로 수정 (예: “나는 BTS를 좋아하는 아미, 소아베야. 코디나 스타일 같이 고민해 보자.”).
- [ ] **말풍선 연출**
  - 채팅 메시지 영역에 **말풍선 + 소아베 얼굴**이 함께 보이도록 레이아웃/컴포넌트 구성 (이미지 경로: 소모오 캐릭터 폴더에서 선택한 소아베 이미지).
- [ ] **이미지 자산**
  - 사용할 소아베 이미지 1~2장을 프로젝트(예: `SIMS_Fashion/image/soave/`)로 복사하거나 경로 통일.

**결과 목표**: 사용자가 채팅을 열면 “소설 속 주인공인 BTS를 좋아하는 아미, 소아베가 말하는 것처럼” 보이기.

### Phase 2: 스토리 구조를 홈에 반영
- [ ] PDF에서 추출한 **장(章)·핵심 장면**을 메뉴 또는 섹션으로 설계 (예: “1화”, “2화” 또는 “프롤로그”, “첫 만남” 등).

- [ ] 기존 **히어로/서비스** 문구를 “소아베 시점” 한 줄 내레이션으로 바꿀지 여부 결정 후 적용.

### Phase 3: 소모오 캐릭터 확장 활용

- [ ] 필요 시 **짧은 영상(mp4)** 삽입 구간 정하고, 재생 UI 추가.
- [ ] 일관된 **말풍선 + 캐릭터 얼굴** 컴포넌트를 재사용할 수 있도록 정리.

### Phase 4: 스토리 전용 페이지/모드 (선택)

## 4. 우선 적용 예시 (Phase 1 구체화)

- **맨 오른쪽 말풍선(플로팅 채팅 버튼)**
  - 현재: “AI” 배지 + 말풍선 아이콘.
  - 변경: **소아베 캐릭터 얼굴** 이미지를 작은 원형으로 넣어, “소아베가 나에게 말 걸어오는 느낌”으로 연출.

- **채팅 창 열었을 때**
  - 상단: **소아베 얼굴** + 이름 “소아베” + 한 줄 소개 (예: “BTS를 사랑하는 아미, 너와 스타일 고민할게.”).
  - 웰컴/첫 메시지: 말풍선 형태 + 같은 소아베 얼굴로 “소아베가 말하는” 톤.
  - 사용자 메시지에 대한 답도 “소아베가 말하는 것처럼” 문체로 (기존 AI 답변을 소아베 1인칭으로 포맷하거나, 프롬프트에 “소아베 톤”을 넣는 방식 검토).

- **이미지 경로 예시 (개발 시)**
  - 로컬: `E:\000000000000_음색마을\소모오 캐릭터\소아베\소아베 이미지\` 중 정면·웃는 얼굴 1장 선택.
  - 프로젝트에 복사: `SIMS_Fashion/image/soave/avatar.png` (또는 `soave-chat.png`) 등으로 두고, HTML/CSS에서 해당 경로 참조.
